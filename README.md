Program Hakkında:

Program kullanıcı tarafından girilen 2 tamsayı için dört işlem yapmaktadır.

Nasıl Çalıştırılır?:

1)İlk etapta yapmak istediğiniz işlemi belritmeniz gerekmektedir,toplama işlemi için "+",çıkarma işlemi için "-",çarpma işlemi için "*",bölme şlemi için "/" girmeniz gerekmektedir.

2)Seçtiğiniz işlemin hangi sayılar için yapılacağını giriniz.

3)Girilen sayılar için yapılan işlemin sonucu ekrada görüntülenecektir.
